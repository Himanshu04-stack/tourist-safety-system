🛡️ Smart Tourist Safety Monitoring & Incident Response System

📌 Overview

The Smart Tourist Safety Monitoring & Incident Response System is an AI-powered, blockchain-enabled platform designed to enhance the safety and security of tourists in high-risk and remote areas.

It integrates AI, Geo-Fencing, Blockchain, and IoT technologies to provide real-time monitoring, emergency response, and secure identity verification while ensuring data privacy.

🎯 Problem Statement

Tourist safety remains a major concern in remote and high-risk regions where traditional monitoring systems are inefficient.

This project aims to build a technology-driven ecosystem that ensures:
	•	Real-time tourist tracking
	•	Quick emergency response
	•	Secure digital identity verification
	•	Predictive risk analysis

🚀 Key Features

🔐 Digital Tourist ID (Blockchain-Based)
	•	Secure and tamper-proof digital identity
	•	Includes:
	•	KYC details (Aadhaar/Passport)
	•	Travel itinerary
	•	Emergency contacts
	•	Valid only for trip duration

📱 Mobile Application for Tourists
	•	Tourist Safety Score (AI-based)
	•	Geo-fencing alerts for restricted zones
	•	🚨 Panic Button with live location sharing
	•	Optional real-time tracking (opt-in)

🤖 AI-Based Anomaly Detection
	•	Detects:
	•	Sudden inactivity
	•	Route deviation
	•	Location drop-offs
	•	Flags potential distress or missing cases

🖥️ Admin Dashboard (Police & Tourism Dept.)
	•	Real-time tourist tracking
	•	Heatmaps of high-risk zones
	•	Digital ID access
	•	Automated E-FIR generation

📡 IoT Integration (Optional)
	•	Smart bands/tags for high-risk areas
	•	Continuous health & location monitoring
	•	Manual SOS trigger

🌐 Multilingual Support
	•	Supports 10+ Indian languages + English
	•	Voice/text emergency access
	•	Accessible for elderly & disabled users

🔒 Data Privacy & Security
	•	End-to-end encryption
	•	Blockchain ensures tamper-proof records
	•	Compliance with data protection laws

🏗️ System Architecture

Tourist App ↔ Backend Server ↔ AI Engine
        ↘ Blockchain Network
        ↘ Police Dashboard
        ↘ IoT Devices (Optional)

⚙️ Installation & Setup

1. Clone the Repository

git clone https://github.com/Himanshu04-stack/tourist-safety-system.git
cd tourist-safety-system

2. Backend Setup

cd backend
mvn clean install
mvn spring-boot:run

3. Frontend Setup

cd frontend
npm install
npm start

4. Mobile App
	•	Open in Android Studio
	•	Sync Gradle
	•	Run on emulator/device
